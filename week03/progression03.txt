function.js
ทำให้รู้ว่า เราสามารถสร้าง ฟังก์ชั่นในรูปแบบไหนบ้าง ทั้ง function declaration ที่เป็นโครงสร้างชัดเจน และ function expression ที่เป็นการเขียนฟังก์ชั่นแบบย่อ ที่ช่วยให้เราเขียนได้สั้นลงมากๆ และก็ได้รู้เรื่องข้อจำกัดของการสร้างแต่ละแบบ

funcArguments.js
เป็น special arguments object ที่จะเก็บข้อมูลทุกอย่าง ที่ส่งมาที่ฟังก์ชั่น โดยไม่จำเป็นต้องสร้างพารามิเตอร์ โดยที่เราสามารถนำมาใช้ประโยชน์ได้

greeting.js
เนื้อหานี้เรากำลังจะทำ ค่า default จาก พารามิเตอร์ โดยที่ พารามิเตอร์ ต่ำแหน่งไหน ที่ไม่ได้รับข้อมูลก็จะได้รับค่า default มาแทน ซึ่ง undefined คือการไม่ให้ข้อมูลกับพารามิเตอร์

getFeqOfWord.js
เป็นการนำเรื่อง special arguments object เข้ามาใช้หลังจากที่พึ่งได้เรียนไป โดยที่ อจ ซึ่งจริงๆแล้ว โจทย์นี้ยังไม่ยากมาก แต่ผมอยากเอาเท่ เขียนโดยใช้ funcional แต่สุดท้าย ผมยังใช้ไม่ค่อยคล่องทำให้ต้องกลับมาเบสิก
